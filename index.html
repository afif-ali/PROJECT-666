PROJECT 666

Description:

PROJECT 666 is a passive agressive virus made with the sole purpose of bothering whoever uses it. After signing a contract with the devil with the help of LARRY, the virus takes over your computer, spamming you with empty windows, getting PETER GRIFFIN to demotivate you, or even throwing random jumpscares when you least expect it. After a while, GOD comes to you and bluescreens your computer... unless, you know the secret passcode to abort. CAN YOUR PC SURVIVE?

Technologies:

PROJECT 666 was made using exclusively PYTHON as a language and PyQt5 and an interface.

How to use:

Download the source code from GITHUB and open a terminal in the root directory. Check that PYTHON and PIP are both downloaded and download the required libraries using the following commands:
pip install PyQt5
pip install pygame
You can now run the virus with the command python main.py
